## 常用输入、输出语句

#### printf()

* 函数原型 

  ```cpp
  #include<cstdio>
  int printf(const char* format, ...); //按照format格式字符串中的格式说明将后面变量输出
  ```

* 格式说明符
  * 类型：`%d`（int）、`%f`（double）、`%c`（char）、`%s`（字符串）
  * 标志：`-`（左对齐）、`+`（显示正负号）、`0`（填充零）
  * 宽度
  * 精度：默认保留6位小数

#### scanf()

* 函数原型

  ```cpp
  #include<cstdio>
  int scanf(const char* format, ...); //按照format字符串中的格式说明符将输入转换为指定类型存入对应地址
  ```

* 格式说明符

  * 类型：类似`printf()`

* 注意事项
  1. 必须传入变量地址
  2. 输入中的空白：`%d`、`%f`、`%s` 会自动跳过前导空白（空格、制表符、换行）；`%c` 不会跳过

#### getchar（）

* 函数原型

  ```cpp
  #include<cstdio>
  int getchar(void);  //从标准输入读取下一个字符，并将其作为 unsigned char 转换为 int 返回;文件末尾或者读取失败返回EOF（-1）
  ```

* 示例：逐字符读取直到换行

  ```cpp
  #include <cstdio>
  int main() {
      int ch;
      printf("请输入一行文字（以回车结束）：");
      while ((ch = getchar()) != '\n' && ch != EOF) {  //常配合循环使用
          putchar(ch);  // 输出读取的字符
      }
      return 0;
  }
  ```

#### putchar()

* 函数原型

  ```cpp
  #include<cstdio>
  int putchar(int ch);
  ```

## `algorithm`常用函数汇总

* 排序相关

  | 函数                                | 说明                                                         | 时间复杂度                |
  | :---------------------------------- | :----------------------------------------------------------- | :------------------------ |
  | `sort(first, last)`                 | 对区间 `[first, last)` 进行升序排序（默认 `<`）              | O(N log N)                |
  | `sort(first, last, comp)`           | 按自定义比较器 `comp` 排序                                   | O(N log N)                |
  | `stable_sort`                       | 稳定排序，相等元素保持原相对顺序                             | O(N log N) 或 O(N log² N) |
  | `partial_sort(first, middle, last)` | 将 `[first, middle)` 部分排序为整个区间最小的元素            | 大致 O(N log M)           |
  | `nth_element(first, nth, last)`     | 将第 `nth` 位置的元素放到排序后应在的位置，且左边都 ≤ 它，右边 ≥ 它 | 平均 O(N)                 |
  
  ```cpp
  vector<int> v = {5, 2, 8, 1, 9};
  sort(v.begin(), v.end());                     // 升序：1 2 5 8 9
  sort(v.begin(), v.end(), greater<int>());     // 降序：9 8 5 2 1
  partial_sort(v.begin(), v.begin()+3, v.end()); // 前3个是最小的三个数
  nth_element(v.begin(), v.begin()+2, v.end()); // v[2] 是第3小的数
  ```



* 查找与搜索

  | 函数                                | 说明                                                  | 要求         |
  | :---------------------------------- | :---------------------------------------------------- | :----------- |
  | `find(first, last, value)`          | 返回第一个等于 `value` 的迭代器，未找到返回 `last`    | 线性查找     |
  | `find_if(first, last, pred)`        | 返回第一个满足谓词 `pred` 的迭代器                    | 线性查找     |
  | `binary_search(first, last, value)` | 判断有序区间是否存在 `value`，返回 `bool`             | 区间必须有序 |
  | `lower_bound(first, last, value)`   | 返回第一个 **不小于** `value` 的位置（即 `>= value`） | 有序区间     |
  | `upper_bound(first, last, value)`   | 返回第一个 **大于** `value` 的位置                    | 有序区间     |

  ```cpp
  vector<int> v = {1, 2, 3, 3, 5};
  auto it = find(v.begin(), v.end(), 3);        // 指向第一个 3
  bool exist = binary_search(v.begin(), v.end(), 3); // true
  auto low = lower_bound(v.begin(), v.end(), 3); // 指向第一个 3
  auto up = upper_bound(v.begin(), v.end(), 3);  // 指向 5（3之后的位置）
  auto range = equal_range(v.begin(), v.end(), 3); // pair(low, up)
  ```



* 计数

  | 函数                          | 说明                              |
  | :---------------------------- | :-------------------------------- |
  | `count(first, last, value)`   | 返回区间中等于 `value` 的元素个数 |
  | `count_if(first, last, pred)` | 返回满足谓词 `pred` 的元素个数    |

  ```cpp
  vector<int> v = {1, 2, 3, 3, 5};
  int cnt = count(v.begin(), v.end(), 3);           // 2
  int even = count_if(v.begin(), v.end(), [](int x){ return x%2==0; }); // 1
  ```



* 最大最小值

  | 函数                          | 说明                                          |
  | :---------------------------- | :-------------------------------------------- |
  | `max(a, b)`                   | 返回较大值（也可用初始化列表 `max({a,b,c})`） |
  | `min(a, b)`                   | 返回较小值                                    |
  | `minmax(a, b)`                | 返回 `pair`（最小值，最大值）                 |
  | `max_element(first, last)`    | 返回指向最大元素的迭代器                      |
  | `min_element(first, last)`    | 返回指向最小元素的迭代器                      |
  | `minmax_element(first, last)` | 返回 `pair`（最小元素迭代器，最大元素迭代器） |

  ```cpp

  auto it = max_element(v.begin(), v.end()); // 指向 5
  int m = max({3,1,4,1,5});                  // 5
  ```



* 修改

  | 函数                                     | 说明                                                         |
  | :--------------------------------------- | :----------------------------------------------------------- |
  | `copy(first, last, result)`              | 复制区间到从 `result` 开始的位置                             |
  | `copy_if(first, last, result, pred)`     | 仅复制满足 `pred` 的元素                                     |
  | `fill(first, last, value)`               | 将区间所有元素赋值为 `value`                                 |
  | `fill_n(first, n, value)`                | 从 `first` 开始连续 `n` 个元素赋值为 `value`                 |
  | `replace(first, last, old_val, new_val)` | 将区间中等于 `old_val` 的元素替换为 `new_val`                |
  | `replace_if(first, last, pred, new_val)` | 将满足 `pred` 的元素替换为 `new_val`                         |
  | `remove(first, last, value)`             | **移除**（逻辑上）等于 `value` 的元素，返回新结尾（注意需配合 `erase` 真正删除） |
  | `remove_if(first, last, pred)`           | 逻辑移除满足条件的元素                                       |
  | `unique(first, last)`                    | 移除连续重复元素，返回新结尾（通常需配合 `erase`）           |

  ```cpp
  v.erase(new_end, v.end());                     // 真正删除
  
  v = {1,1,2,2,3,3};
  new_end = unique(v.begin(), v.end());          // 去重，结果：1,2,3
  v.erase(new_end, v.end());
  ```

## `numeric`常用函数汇总

* `accumulate`累加

  **函数原型**：

  ```cpp
  // 版本1：默认加法累加
  T accumulate(InputIt first, InputIt last, T init);
  
  // 版本2：自定义二元操作
  T accumulate(InputIt first, InputIt last, T init, BinaryOperation op);
  ```

  - **作用**：对区间 `[first, last)` 的元素进行累加（或自定义操作），初始值为 `init`。
  - **返回值**：累加结果，类型与 `init` 相同。
  - **注意**：默认使用 `operator+`，注意类型转换和溢出。

  **示例**：

  ```cpp
  #include <numeric>
  #include <vector>
  #include <iostream>
  
  int main() {
      std::vector<int> v = {1, 2, 3, 4, 5};
  
      // 求和：1+2+3+4+5+0 = 15
      int sum = std::accumulate(v.begin(), v.end(), 0);
  
      // 求乘积：1*2*3*4*5*1 = 120
      int prod = std::accumulate(v.begin(), v.end(), 1, std::multiplies<int>());
  
      std::cout << "sum=" << sum << ", prod=" << prod << '\n'; // sum=15, prod=120
      return 0;
  }
  ```



* `gcd`最大公约数

  **函数原型**：

  ```cpp
  constexpr std::common_type_t<M, N> gcd(M m, N n);
  ```

  - **作用**：计算 `|m|` 和 `|n|` 的最大公约数（总是返回非负值）。
  - **参数**：整数类型（包括 `bool`、`char` 等，但不包括浮点）。
  - **返回**：若 `m` 和 `n` 均为 0，返回 0；否则返回非负 gcd。

  **示例**：

  ```cpp
  int g = std::gcd(12, 18); // 6
  int g2 = std::gcd(-12, 18); // 6
  int g3 = std::gcd(0, 5); // 5
  ```



* `lcm`最小公倍数

  **函数原型**：

  ```cpp
  constexpr std::common_type_t<M, N> lcm(M m, N n);
  ```

  - **作用**：计算 `|m|` 和 `|n|` 的最小公倍数（非负值）。
  - **注意**：当结果超出整数范围时行为未定义，因此需注意溢出风险。

  **示例**：

  ```cpp
  int l = std::lcm(12, 18); // 36
  int l2 = std::lcm(0, 5); // 0（按定义，0 与任何数的 lcm 为 0）
  ```

  > **提示**：在 GCC 等编译器中，常用非标准 `__gcd` 和自定义 `lcm`，但 C++17 标准已提供 `std::gcd` 和 `std::lcm`，建议优先使用标准版本。

## 字符串操作

### C风格字符串

C 风格字符串本质上是以空字符 `'\0'` 结尾的字符数组。它们源于 C 语言，在 C++ 中依然广泛使用，尤其在与系统接口或老旧代码交互时。

### 1. 头文件

```cpp
#include <cstring>   // C++ 中推荐
#include <string.h>  // C 语言方式
```

### 2. 常用函数

| 函数                                   | 说明                                                         | 注意事项                                               |
| :------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------- |
| **`strlen(s)`**                        | 返回字符串长度（不含 `'\0'`）                                | 时间复杂度 O(n)                                        |
| **`strcpy(dest, src)`**                | 将 src 复制到 dest                                           | **危险**：不检查 dest 容量，可能缓冲区溢出             |
| **`strncpy(dest, src, n)`**            | 最多复制 n 个字符到 dest，若 src 长度 < n，剩余填 `'\0'`     | 若 src 长度 ≥ n，dest 可能不以 `'\0'` 结尾，需手动添加 |
| **`strcat(dest, src)`**                | 将 src 追加到 dest 末尾                                      | 同样危险，不检查容量                                   |
| **`strncat(dest, src, n)`**            | 最多追加 n 个字符，并自动添加 `'\0'`                         | 相对安全                                               |
| **`strcmp(s1, s2)`**                   | 比较字符串，返回 0（相等）、负数（s1 < s2）、正数（s1 > s2） | 基于字典序                                             |
| **`strncmp(s1, s2, n)`**               | 比较前 n 个字符                                              | 安全比较                                               |
| **`strchr(s, c)`**                     | 返回指向 s 中第一次出现字符 c 的指针，未找到返回 `nullptr`   |                                                        |
| **`strrchr(s, c)`**                    | 返回指向 s 中最后一次出现字符 c 的指针                       |                                                        |
| **`strstr(s, sub)`**                   | 返回指向 s 中第一次出现子串 sub 的指针，未找到返回 `nullptr` |                                                        |
| **`strtok(s, delim)`**                 | 分割字符串，**非线程安全**，会修改原字符串                   | 使用需谨慎，通常用 `std::string` + 字符串流代替        |
| **`sprintf(buf, format, ...)`**        | 格式化输出到字符数组                                         | 同 `printf`，但有缓冲区溢出风险，建议用 `snprintf`     |
| **`snprintf(buf, size, format, ...)`** | 安全格式化输出，最多写 size-1 个字符，并保证 `'\0'` 结尾     | **推荐使用**                                           |
| **`sscanf(s, format, ...)`**           | 从字符串读取格式化数据                                       |                                                        |
| **`memcpy(dest, src, n)`**             | 复制 n 个字节，不关心 `'\0'`                                 | 用于原始内存操作                                       |
| **`memmove(dest, src, n)`**            | 同 `memcpy`，但处理重叠区域安全                              |                                                        |

### 3. 示例代码

```cpp
#include <cstdio>
#include <cstring>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    char buffer[50];

    // 长度
    printf("Length of str1: %zu\n", strlen(str1));  // 5

    // 复制
    strcpy(buffer, str1);          // buffer = "Hello"
    strcat(buffer, " ");            // buffer = "Hello "
    strncat(buffer, str2, 3);       // buffer = "Hello Wor" (只追加前3个字符)
    printf("buffer: %s\n", buffer);

    // 比较
    if (strcmp(str1, str2) < 0) {
        printf("\"%s\" < \"%s\"\n", str1, str2);
    }

    // 查找字符
    char *p = strchr(str1, 'l');
    if (p) printf("Found 'l' at offset %td\n", p - str1);  // 2

    // 查找子串
    p = strstr(str1, "ll");
    if (p) printf("Found \"ll\" at offset %td\n", p - str1); // 2

    // 安全格式化
    snprintf(buffer, sizeof(buffer), "%s + %s = %s", str1, str2, "HelloWorld");
    printf("%s\n", buffer);  // "Hello + World = HelloWorld"

    return 0;
}
```

### 4. 安全性提示

- **永远不要使用 `gets()`**，它已被 C11 标准移除。
- 使用 `strncpy` 时注意手动添加 `'\0'`：`dest[sizeof(dest)-1] = '\0';`
- 优先选择 `snprintf` 而不是 `sprintf`。
- 避免使用 `strtok`（线程不安全且破坏原串），可用 `std::istringstream` 或手动解析。

## 闰年

**标准定义**：普通闰年：能被4整除但不能被100整除

​                    超级闰年：既能被100整除也能被400整除

**化简定义**：能被400整除或者能被4整除但是不能被100整除

**逻辑表达式**：`(x%400)==0||(x%4==0&&x%100!=0)`
